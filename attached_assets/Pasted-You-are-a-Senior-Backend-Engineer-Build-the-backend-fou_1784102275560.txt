--------------------------------------------------
DATABASE
--------------------------------------------------

For development, use an in-memory MongoDB database (mongodb-memory-server).

However, architect the project so switching to MongoDB Atlas later only requires changing the MONGODB_URI environment variable.

Create a proper database configuration layer.

If MONGODB_URI exists:

→ Connect to Atlas.

Otherwise:

→ Automatically start mongodb-memory-server.

No code elsewhere should know which database is being used.
